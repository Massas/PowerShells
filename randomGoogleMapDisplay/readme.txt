★このスクリプトの概要
　Abstract
　
　このPowerShellスクリプトは、ランダムに緯度・経度を生成して、
　ブラウザを起動、Google Mapに自動生成した地点を表示します。
　
　This PowerShell script can create randomly latitude and longitude,
　display the site indicated latitude & longitude on Google Map

★このスクリプトの実行方法
　How to use this PowerShell script!

１）このリポジトリのファイルをダウンロードする
　　Download this repository to your computer.

２）randomGoogleMapDisplay.batのファイルパス(C:\work\PowerShell\randomPlay_reverseGeocode\randomGoogleMapDisplay.ps1)をご自分の環境に配置したパスに修正して下さい。
　　fix file path below in randomGoogleMapDisplay.bat.
powershell -ExecutionPolicy RemoteSigned -File C:\work\PowerShell\randomPlay_reverseGeocode\randomGoogleMapDisplay.ps1

３）randomGoogleMapDisplay.batをダブルクリックして実行します
　※Windows PowerShellコンソール上では、randomGoogleMapDisplay.ps1が配置しているフォルダに移動して
　　「.\randomGoogleMapDisplay.ps1」で実行できます。
